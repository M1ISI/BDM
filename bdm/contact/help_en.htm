Welcome to ISI Search !

This is a help page to help you to use our new search engine. It offers more type of reasearch than a mere search engine.

1) Run a search

to run a search enter a keyword in the searchbar (1), and select your language (2). The results will automatically display under the search bar and will be classified in function of the search type (3).

2) Different search types

ISI Search offers 4 different search types, each one is represented by a different tree.

a) Apple Tree or classical search

The classical search is in the pink column. It has two parts :

  • The first one (1) is a keywords list which can help you in your search, you just need to click on it to add it to your search.
  • The second one (2) is a websites list which correspond to your search.

b) Bush or exotic search

The exotic search is in the purple column. It aims to guide your search in a less common search with a keywords list more or less linked to your search. To add one of these kewords to your search you just need to click on it.

c) Willow or Social Search

The social search, it is to say a search trought social networks, you can access it with the "accès recherche via réseaux sociaux" button. It will open a new window in witch you can connect on some social networks. You can use the search bar to run a search throught those you are log on.

d) Fir tree or Media search

The media search is in the dark green column. It will return a media list (texts and sounds) wich correspond to you search from our database.

For image search you need to clic on the cat and on the "image" button in the new opened window. It will offer you two way for your search :
First, you can upload an image from your computer and it will return images with similar colors. You can filter black and/or white to ignore background colors.
Sencond, you can choose a color and an error range to get images which contains colors in this range.

By cliking on the cats you can upload texts and musics to enhance the test database.

3) Filter Results

All types of search are display by default. To show/hide one of them you just need to click on the correspondant tree. The tree of a hide research is colored in gray.

4) Help ISI Search developpment

if you find a bug, have a question, use problem, suggestion, remark or anything else to communicate to the developpers, you can send a message whith the form on contact page or directly on our github page.
